# Business and Person Search UI (bor-ui)

## Project configuration
- Node 24
- Nuxt 4
- Nuxt/ui
- Tailwind
- Typescript
- Playwright
- Vitest


## Project setup
```
pnpm install
```

### Compiles and hot-reloads for development
```
pnpm dev
```

### Compiles for production
```
pnpm generate
```

### Locally preview production build
```
pnpm preview
```

### Run e2e tests
```
pnpm test:e2e
```

### Run unit tests
```
pnpm test:unit
```

### Lint check
```
pnpm lint
```
