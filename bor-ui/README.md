# Nuxt 3 Minimal Starter

# bor-ui

## Project configuration
- node 20
- nuxt 3
- Vite
- Vue 3
- Vuetify 3
- Tailwind
- Typescript
- Vitest
- Eslint


## Project setup
```
pnpm install
```

### Compiles and hot-reloads for development
```
pnpm dev
```

### Compiles for production
```
pnpm generate
```

### Locally preview production build
```
pnpm preview
```

### Run your unit tests
```
pnpm test:unit
```

### Lints and fixes files
```
pnpm lint
```
